Quaestor : Your personal expence tracker (backend)

in order to run project on your local 

1 -> npm i (install all the dependencies from json)
2-> deploy a local instance of mongo compass or pass MONGODB_URI in .env to communicate.
3-> then run "node index.js"
if any change made after step 3 , kill the server and run again.
